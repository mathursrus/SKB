MOCK — Issue #30: Google Maps Integration
This mock shows the queue page as accessed from a Google Maps listing. The page is visually identical to the current queue page. The key changes are in the <head> section (meta tags, structured data) which make the page discoverable and previewable on Google Maps and social platforms.
New <head> elements added:
1. <meta name="description"> — SEO description
2. <link rel="canonical"> — canonical URL
3. <meta property="og:*"> — Open Graph tags (4 tags)
4. <meta name="twitter:*"> — Twitter Card tags
5. <script type="application/ld+json"> — Restaurant schema with ReserveAction

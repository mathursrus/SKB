Non-goals for this issue

Intentionally not in scope
· No call recording — continues to comply with SKB's no-record policy
· No conversational AI (Slang.ai / Goodcall / PopMenu) — deferred to a later spec
· No per-day hours logic in the IVR — "seven days a week" is a single static string
· Menu branch does not read individual items — directs to the website for item-level detail
· No multi-language support — English only for this release

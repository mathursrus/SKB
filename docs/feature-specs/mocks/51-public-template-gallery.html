What's shared across templates

  • Same five pages: home / menu / about / hours / contact.
  • Same structured content fields — switching templates preserves data.
  • Same "Join the Waitlist" CTA pattern leading to the diner queue (same experience for diners regardless of the restaurant's template choice).
  • Same fallbacks: no menu uploaded ⇒ "Menu coming soon"; no hero image ⇒ template's built-in gradient placeholder; no about copy ⇒ template's default blurb.
  • Same SEO hooks: JSON-LD restaurant schema, Open Graph tags, canonical URLs generated from publicUrl.

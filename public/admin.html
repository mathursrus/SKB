Your connection — paste into your AI client; the PIN is also on your door-poster PDF

One command in your terminal. Claude remembers the connection for every future chat.

Open Claude Code and ask "What's on my OSH waitlist right now?"

Paste into claude_desktop_config.json (Settings → Developer → Edit Config) and restart Claude Desktop.

In ChatGPT, create a Custom GPT and add a Connector. Paste the endpoint URL above; under "Authentication" choose Custom header and add both headers from this card. (ChatGPT MCP support is rolling out — some accounts see this option under Tools → Actions.)

Working very close to home? You can also run Claude Code on your phone and point it at OSH from anywhere on the floor.

Safety note: the bearer token above is your host PIN. Anyone with it can operate your waitlist. Rotate it from Front desk → Device PIN if you think it's leaked.

Device PIN

The 4–6-digit PIN your hosts type on the tablet to unlock the host stand.

Changing the PIN doesn’t log out tablets already unlocked, but future unlocks require the new value. The PIN is also printed on the door-poster PDF.

Sender display name

Every text to your guests starts with this name so they know which restaurant it’s from. Keep it short and recognizable.
